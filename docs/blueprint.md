# **App Name**: KhetGenie

## Core Features:

- AI Price Oracle: Provide farmers with AI-driven minimum and maximum price recommendations per kg, based on crop type, region, seasonality, and market trends. It uses a reasoning tool to consider the mentioned data.
- Quality Inspector: Analyze uploaded crop images using AI to determine a quality grade (A, B, or C), identify visible issues, and provide a summary report for both farmers and buyers.
- Location Wizard: Convert city names to geographic coordinates using the Open-Meteo Geocoding API, facilitating location-based insights.
- Air Sentinel: Provide insights into local air quality, pollution levels using Open-Meteo Air Quality API and relevant health advice based on geographic coordinates.
- Terms Summarizer: Summarize the agreed terms between farmers and buyers from conversation data, extracting price per kg, quantity, conditions, and highlighting potential misunderstandings.
- Interactive Mapping: Show location and AQI information on the same map.
- Streamlined Listing Form: Create an intuitive, user-friendly listing form with real-time validation, auto-suggestions, and guided input fields to capture crop details accurately.

## Style Guidelines:

- Primary color: Earthy green (#6B8E23) to reflect agriculture and growth.
- Background color: Light beige (#F5F5DC), offering a neutral, natural backdrop.
- Accent color: Warm orange (#D2691E) for call-to-action buttons and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif that blends modernity with a touch of personality.
- Use simple, clean line icons representing crops, locations, and market indicators.
- Implement a grid-based layout to organize marketplace data. Highlight listings through well-defined visual hierarchy.
- Incorporate subtle transitions and animations, for example, map markers appearing with a gentle fade-in, or data cards expanding smoothly.
